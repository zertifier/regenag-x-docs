# Analítica Regional de CO₂ y Benchmarking (C2D)

## Propósito del Algoritmo

Este algoritmo agrega mediciones de CO₂ a nivel regional para crear una huella ambiental estadísticamente robusta por región. Permite la comparación entre regiones preservando la confidencialidad de los sensores y granjas individuales.

Todo el procesamiento ocurre dentro de un contenedor seguro de Compute-to-Data.

## Qué hace el Algoritmo

### Agregación regional
- **Agrupa mediciones de CO₂ por región**
- **Calcula estadísticas globales de primer orden:**
  - media, mediana
  - mín / máx
  - percentiles P05 / P95
  - desviación estándar

### Cobertura temporal
- **Identifica rangos de fechas de medición por región**
- **Detecta lagunas y cobertura irregular**

### Variabilidad y estabilidad
- **Mide la dispersión y volatilidad de los valores de CO₂**
- **Identifica regiones inusualmente estables o inestables**

### Comparación entre regiones (listo para el futuro)
- **Los resultados normalizados permiten la comparación a medida que se añaden más regiones**
- **Permite el benchmarking y la agrupación (clustering)**

## Valor en un flujo de trabajo Compute-to-Data
- Permite analíticas multiregionales sin fuga de datos
- Crea KPIs ambientales regionales reutilizables
- Soporta productos de benchmarking anonimizados

## Valor para la industria
Útil para:
- Plataformas de monitoreo ambiental
- Informes regulatorios
- Selección de ubicación y evaluación de impacto
- Conjuntos de datos regionales de CO₂ listos para el mercado
