# Analítica de CO₂ y Benchmarking a Nivel de Sensor (C2D)

## Propósito del Algoritmo

Este algoritmo analiza el comportamiento del CO₂ a nivel de sensor individual para detectar patrones de contaminación anormales, deriva del sensor y desviaciones relativas entre sensores que operan en entornos similares.

Está diseñado para soportar tanto el conocimiento ambiental como el aseguramiento de la calidad de los sensores.

## Qué hace el Algoritmo

### Estadísticas de CO₂ por sensor
- media, mediana
- desviación estándar
- índice de volatilidad
- mín / máx
- detección de picos P01 / P99

### Benchmarking relativo
- **Compara los niveles de CO₂ de cada sensor con la mediana global**
- **Identifica sensores que reportan valores consistentemente más altos o más bajos**

### Detección de anomalías
- **Detecta picos de CO₂ y valores extremos**
- **Marca sensores con varianza excesiva**

### Separación de señal de calidad vs. entorno
- **Ayuda a distinguir la contaminación ambiental real del mal funcionamiento del sensor**

## Valor en un flujo de trabajo Compute-to-Data
- Permite la diferenciación entre granja sucia vs. sensor estropeado
- Produce resultados analíticos compactos y monetizables
- Seguro para conjuntos de datos agrícolas o industriales confidenciales

## Valor para la industria
Soporta:
- Detección de puntos calientes de contaminación
- Auditorías de calibración de sensores
- Certificación ambiental
- Productos de benchmarking avanzados
