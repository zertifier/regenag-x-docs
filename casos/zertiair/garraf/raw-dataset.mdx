# Dataset de CO2 al Aire Libre (Raw) - Garraf

## Descripción General

ZertiAir proporciona un conjunto de datos descargable de mediciones continuas de CO₂ al aire libre recopiladas de sensores desplegados en la región del Garraf (Megatró). El conjunto de datos se puede acceder "tal cual" y contiene observaciones de series temporales de CO₂ adecuadas para el análisis independiente, la investigación y la integración en sistemas externos.

## Características del Dataset

### Tipo de Acceso
**Dataset descargable – "As Is"**

Los datos están disponibles para descarga directa sin procesamiento adicional, permitiendo a los usuarios realizar sus propios análisis y transformaciones según sus necesidades específicas.

### Contenido del Dataset

El dataset contiene las siguientes mediciones y metadatos:

| Campo | Descripción | Tipo | Ejemplo |
|-------|-------------|------|---------|
| **farm** | Nombre de la bodega (puede ser "Anonymous") | Alfanumérico | "Anonymous" |
| **region** | Comarca o región geográfica | Alfanumérico | "Garraf" |
| **sensorId** | Identificador único del sensor | Alfanumérico | "24E124126C143207" |
| **co2ppm** | Concentración de CO₂ en ppm | Numérico | 718 |
| **sensorBatteryPercentage** | Porcentaje de batería del sensor | Numérico | 95 |
| **createdAt** | Fecha y hora del registro | Timestamp | "2025-11-07 20:52:33" |

### Ejemplo de Registro

```json
{
  "farm": "Anonymous",
  "region": "Garraf",
  "sensorId": "24E124126C143207",
  "co2ppm": 718,
  "sensorBatteryPercentage": 95,
  "createdAt": "2025-11-07 20:52:33"
}
```

## Características Técnicas

### Formato y Estructura
- **Formato**: JSON normalizado
- **Tamaño medio**: 214 KB
- **Frecuencia de actualización**: Mensual
- **Región**: Garraf (Megatró)

### Tecnología de Captura
- **Sensores**: Zertiair CO₂ IoT
- **Protocolo**: LoRaWAN
- **Certificación**: Dispositivos certificados para entornos industriales/vinícolas
- **Alimentación**: Optimizada para funcionamiento continuo

## Privacidad y Anonimización

El dataset ha sido procesado siguiendo estrictos protocolos de anonimización:

### Medidas Aplicadas
- ✅ Eliminación de identificadores directos (nombres de propietarios)
- ✅ Generalización de localización (nivel de comarca/región)
- ✅ Uso exclusivo de identificadores técnicos no personales
- ✅ Cumplimiento RGPD y LOPDGDD

### Nivel de Protección
Los datos son **completamente anonimizados** y no tienen la consideración de datos personales a efectos del RGPD.
