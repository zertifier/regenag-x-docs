Multi-Winery CO2 Analytics (Compute-to-Data) - Garraf

Overview​

Exclusive access to outdoor CO₂ measurements from wineries in the Garraf region (Megatró) through Compute-to-Data algorithms. Instead of downloading raw sensor data, users obtain privacy-safe analytics, including sensor health metrics, regional CO₂ footprints, day-night temporal behavior, and anomaly detection.

Dataset Features​

Access Type​

Computed Asset – Algorithm Only

The data is not downloaded directly. Instead, users run authorized algorithms that process the data in-situ and return only the analytical results.

Compute-to-Data Paradigm (C2D)​

This approach allows outdoor CO₂ to function as a reliable boundary condition for advanced models without exposing raw measurements, enabling secure reuse and monetization of environmental intelligence.

Available Algorithms​

The dataset includes access to five free algorithms for enrichment and analysis:

1. Sensor Inventory & Battery Health Analytics​

Sensor Inventory and Health Analytics

Analyzes sensor population, sampling behavior, battery performance, and detects abnormal discharge patterns - entirely within a Compute-to-Data environment.

Generated outputs:

  • Complete inventory of active sensors
  • Sampling frequency statistics
  • Battery health analysis
  • Abnormal discharge pattern detection
  • Preventive maintenance prediction

View Algorithm Details →

2. Temporal CO₂ Analytics (Day/Night & Trends)​

Temporal CO₂ Analytics

Analyzes diurnal patterns, trends, and temporal anomalies in CO₂ data for sensors and regions.

Generated outputs:

  • Day/night CO₂ concentration patterns
  • Short and long-term temporal trends
  • Cycle and periodicity identification
  • Temporal anomaly detection
  • Period-to-period comparisons

View Algorithm Details →

3. Regional CO₂ Analytics & Benchmarking​

Regional Analytics and Benchmarking

Calculates regional CO₂ statistics, trends, and variability to enable secure multi-location comparisons without exposing raw sensor data.

Generated outputs:

  • Regional CO₂ footprints (averages, medians, percentiles)
  • Anonymized inter-regional comparisons
  • Regional variability analysis
  • Geographic outlier identification
  • Air quality rankings by region

View Algorithm Details →

4. Sensor-Level CO₂ Analytics & Pollution Benchmarking​

Sensor-Level Analytics

Detects high-pollution sensors, volatility, and abnormal behavior through secure per-sensor CO₂ analytics.

Generated outputs:

  • Identification of sensors with high CO₂ concentration
  • Per-sensor volatility analysis
  • Inter-sensor pollution comparisons
  • Atypical behavior detection
  • Individual performance metrics

View Algorithm Details →

5. Advanced CO₂ & Sensor Anomaly Detection​

Advanced Anomaly Detection

Identifies extreme events, spikes, drift, and second-order anomalies across sensors and regions.

Generated outputs:

  • Extreme CO₂ event detection
  • Abnormal spike identification
  • Sensor drift analysis
  • Second-order anomalies (pattern changes)
  • Early malfunction warnings
