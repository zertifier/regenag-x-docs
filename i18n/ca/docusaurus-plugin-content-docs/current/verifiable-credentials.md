# Credencials Verificables i Gaia-X

Les Credencials Verificables (VCs) permeten demostrar atributs o fets (per exemple, identitat legal, rols o conformitat) de manera criptogràficament segura, verificable sense contactar amb l'emissor i amb control per part del titular. A Gaia-X, les VCs i les Autodescripcions signades són la base del marc de confiança per identificar participants, datasets i serveis.

Per què és bo per a l'automatització i la descobribilitat per IA?

- Confiança automatitzable: els validadors poden comprovar signatures i l'estat de conformitat sense intervenció humana.
- Descobribilitat semàntica: les autodescripcions en format estàndard faciliten que catàlegs i agents d'IA trobin i comparin actius.
- Menys fricció: l'onboarding i el compliment s'acceleren amb verificacions repetibles (p. ex., GXDCH Compliance i Notary).

Al nostre espai de dades, les VCs donen suport a l'alta d'organitzacions, la publicació de datasets i l'exposició d'algoritmes de confiança, possibilitant fluxos CI/CD que rebutgen actius no conformes.
