# Multi-Farm CO2 Analytics (Compute-to-Data) - Pla de l'Estany

## Overview

Exclusive access to outdoor CO₂ measurements from multiple farms through Compute-to-Data algorithms. Instead of downloading raw sensor data, users obtain privacy-safe analytics, including sensor health metrics, regional CO₂ footprints, day-night temporal behavior, and anomaly detection.

## Dataset Features

### Access Type
**Computed Asset – Algorithm Only**

The data is not downloaded directly. Instead, users run **authorized algorithms** that process the data in-situ and return only the analytical results.

### Compute-to-Data Paradigm (C2D)

This approach allows outdoor CO₂ to function as a **reliable boundary condition** for advanced models without exposing raw measurements, enabling secure reuse and monetization of environmental intelligence.
