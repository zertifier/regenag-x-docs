# Service Offering

## What is a Service Offering?

In the context of data spaces and the international Gaia-X standard, a **Service Offering** is a structured, formal, and standardised description of a service, resource, or digital asset that a participant makes available to the ecosystem.

This description acts as a "technical sheet" or a digital passport for a service. It contains all the technical, legal, privacy, and commercial details (such as terms, policies, and technical capabilities, e.g. an API or an algorithm) that any user needs to discover it, understand its conditions of use, and consume it with full confidence.

In a decentralised data environment, the Service Offering is the key piece that enables services to be discovered and contracted in an automated and secure way.

## How do we apply it in Regenag-x?

In **Regenag-x**, technical trust and certification are fundamental pillars of our architecture. We apply the Service Offering concept directly to our data processing infrastructure.

Specifically, **for each data analysis algorithm (Compute-to-Data) we create, we automatically generate an associated Service Offering.**

This process is critical because:

1. **It standardises our capabilities**: Packaging the algorithm's description as a Service Offering ensures we "do things" using the same language and format as the rest of the participants in the European data network.
2. **It enables Certification (GXDCH)**: By having this Service Offering correctly structured, we can submit it to a **Gaia-X Digital Clearing House (GXDCH)**. This entity acts as an independent technical validator that reviews the algorithm's description.
3. **It generates Compliance Credentials**: After validation, the GXDCH issues us a **Gaia-X compliance certificate** (a Verifiable Credential). This certificate guarantees all other actors (farmers, cooperatives, researchers) that the algorithm used in Regenag-x is secure, reliable, and operates in strict compliance with European data trust and sovereignty standards.

Thanks to this process, any farmer or cooperative using Regenag-x computing has the verifiable guarantee that their data is being analysed by technological services certified under the highest European standards.
